# Progress Reports

Store local progress reports for multi-step work in this folder.

Suggested pattern:

- one Markdown file per active effort
- prefix each filename with the current branch tip short hash, e.g. `8a229da-2026-04-16-market-data-traceability.md`
- update it when scope, blockers, decisions, or next steps change materially
- prune or archive stale files locally when the work is finished

These progress reports are intentionally gitignored.
