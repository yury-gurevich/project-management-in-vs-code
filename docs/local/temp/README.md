# Temporary Scratch

Short-lived outputs live here: one-off reviews, research notes, scratch
analyses, agent by-products, anything that shouldn't survive a sprint closeout.

Conventions:

- Default destination for scratch work and one-off reviews.
- Empty this folder as part of the sprint-loop cleanup (see
  [docs/sprint-loop.md](../../sprint-loop.md)).
- Nothing in here is authoritative. If something becomes load-bearing, promote
  it to `docs/local/` (runbook) or `docs/` (public).
- Gitignored except this README.
