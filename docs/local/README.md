# Local Working Records

This folder is reserved for local-only planning and workflow records.

Tracked repository-facing docs should not expose internal process details, but
local working notes may still live here for day-to-day use.

Use this folder for:

- long-lived runbooks and operator notes (analyst, override-observation, etc.)
- `progress/` — per-effort progress reports, hash-prefixed, deleted when shipped
- `temp/` — short-lived scratch: one-off reviews, research, agent by-products;
  emptied at sprint closeout

Everything in this folder is gitignored except this note,
[progress/README.md](progress/README.md), and
[temp/README.md](temp/README.md).
